Hello!

To begin this project:

Open a new terminal and type the following commands:
cd littlelemon
pipenv shell
pipenv install
python3 manage.py makemigrations
python3 manage.py migrate
Go to the link http://127.0.0.1:8000/ to see the Little Lemon website
