Hello!

To begin this project:

1. Open a new terminal and type the following commands:
   i. cd littlelemon
   ii. pipenv shell
   iii. pipenv install
   iv. python3 manage.py makemigrations
   v. python3 manage.py migrate
   
3. Go to the link http://127.0.0.1:8000/ to see the Little Lemon website
